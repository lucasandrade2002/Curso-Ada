CURSO ADA (MODULO II)

CÁLCULO BÁSICO
 
 FUNÇÕES
  SÃO RELAÇÕES ENTRE DOIS OU MAIS CONJUNTOS DE INFORMAÇÕES.

  EX: A = {1,2,3}, B = {4,9,0} ===> A & B = {(1,4),(1,9),(1,0),(2,4)...}

  TAMBÉM DENOMINAMOS ESSA RELAÇÃO DE PRODUTO CARTESIANO.

  A RELAÇÃO DE UM CONJUNTO A X B SE DÁ PELA FORMAÇÃO DE UM NOVO SUBCONJUNTO DENTRE TODAS AS   POSSIBILIDADES DE COMBINAÇÃO DE A & B.

  EX: R C A X B = {(1,4),(2,4)}

 DOMÍNIO DE UMA FUNÇÃO:
  TODOS OS ELEMENTOS DE UM CONJUNTO A.

 CONTRA-DOMÍNIO:
  TODOS OS ELEMENTOS DE UM CONJUNTO B.

 IMAGEM DA FUNÇÃO:
  TODOS OS ELEMENTOS DE UM CONJUNTO B QUE CONTÉM RELAÇÃO COM O CONJUNTO A.

 FUNÇÕES INJETORAS:
  FUNÇÕES EM QUE TODO ELEMENTO DE UM CONJUNTO A CONTÉM RELAÇÃO COM UM ELEMENTO DIFERENTE EM   B.

 FUNÇÕES SOBREJETORAS:
  FUNÇÕES EM QUE MAIS DE UM ELEMENTO DE UM CONJUTO A PODE CONTER RELAÇÃO COM O MESMO   ELEMENTO DE UM CONJUNTO B.

EXPRESSÃO FUNCIONAL
 MÉTODO NO QUAL OBTEMOS A FÓRMULA DA FUNÇÃO, DE MODO A OBTER A IMAGEM DELA, A PARTIR DO   DOMÍNIO E LEVANDO-A A PARTIR DA FÓRMULA PARA O CONTRA-DOMÍNIO. 

 EX: F(X) = 2X+1 ===> F(9) = 2.(9) + 1 = 19

 A PARTIR DE UMA EXPRESSÃO FUNCIONAL, CONSEGUIMOS OBTER UM PRODUTO CARTESIANO, ONDE:

 EIXO X: É O ELEMENTO DO CONJUNTO A, OU SEJA, X.
 EIXO Y: É O RESULTADO OBTIDO A PARTIR DA FÓRMULA, OU SEJA, F(X).

 FUNÇÕES QUADRÁTICAS, OU SEJA, QUE CONTÉM UM ELEMENTO A ELEVADO A 2 CONTÉM O COMPORTAMENTO   DE UMA PARÁBOLA.

 CARACTERÍSTICAS DE FUNÇÕES ALGÉBRICAS

  PARIDADE: F(X) = F(-X)
  IMPARIDADE: F(X) = -F(X)

FUNÇÃO INVERSA
 É O MÉTODO QUE MAPEIA ELEMENTOS DO CONTRA-DOMÍNIO PARA O DOMÍNIO, DADA PELA NOMENCLATURA:  F-1(X).

 PODEMOS DIZER QUE A FUNÇÃO INVERSA É O ESPELHO DA EXPRESSÃO FUNCIONAL EM UM GRÁFICO  CARTESIANO.

 F(X) = 3X + 2

 Y = 3X + 2
 X = 3Y + 2
 X - 2 = 3Y
 (X - 2)/3 = Y
 F-1(X) = (X-2)/3 

 FUNÇÃO LINEAR
  É DADA PELA FÓRMULA: F(X) = AX + B

 FUNÇÃO QUADRÁTICA
  É DADA PELA FÓRMULA; F(X) = AX² + BX + C

 FUNÇÃO RACIONAL
  É DADA PELA RAZÃO ENTRE DUAS FUNÇÕES: F(X) / P(X)

 FUNÇÃO TRIGONOMÉTRICA
  É DADA PELO COSSENO, SENO E TANGENTE DE UM ELEMENTO.
